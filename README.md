# PiMao ETF 投资分析系统

PiMao ETF 投资分析系统是一个基于多Agent协作的自动化ETF投资分析平台。

## 📱 移动端报告归档

本项目包含适配手机浏览的HTML报告，方便微信转发和移动端阅读。

### 访问地址

GitHub Pages: `https://tjkj10.github.io/pimaoETF/`

### 目录结构

```
├── index.html                    # 主页面（日期导航）
├── README.md                     # 项目说明
└── reports/                      # 报告归档
    └── 20260208/                 # 2026年2月8日报告
        ├── index.html            # 该日期入口
        ├── decision.html         # 投资决策报告
        ├── sector.html           # 板块分析报告
        ├── macro.html            # 宏观分析报告
        └── review.html           # 复盘分析报告
```

### 特性

- 📱 **响应式设计** - 完美适配手机屏幕（320px - 750px）
- 💬 **微信优化** - 专为微信内置浏览器优化
- 📊 **表格滚动** - 支持横向滚动查看完整表格
- 🎨 **美观界面** - 卡片式布局，视觉层次清晰
- 📅 **日期导航** - 方便查看历史报告

## 📝 添加新日期报告

1. 创建日期目录：`mkdir reports/YYYYMMDD`
2. 复制HTML报告到该目录
3. 更新 `index.html` 中的日期列表
4. 提交到GitHub
